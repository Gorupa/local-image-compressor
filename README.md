# Local Image Compressor 🗜️

A lightning-fast, privacy-friendly, and 100% local image compressor built with pure HTML, CSS, and JavaScript. 

## 🛡️ Privacy First
Most free compression tools upload your personal photos to remote servers. This tool uses the native HTML5 Canvas API to compress your files **entirely inside your browser**. Your files never leave your device. 

## ✨ Features
- **Zero Server Uploads:** 100% client-side processing.
- **Ad-Free & Open Source:** Clean, distraction-free Material Design interface.
- **Live Quality Control:** Adjust the compression ratio on the fly.
- **Format Support:** Compresses JPG, JPEG, and PNG files.
